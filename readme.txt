WooCommerce Order status notifications send using Divine Info SMS Service.

== Description ==

WooCommerce SMS by Divine Info Service plugin will allow you to send SMS notification automatically for orders placed in WooCommerce. Buyer and Admin both can get SMS notification after WooCommerce order is placed. All SMS Notifications text & settings can be customized from wordpress admin panel.

This plugin adds country flag and ISD code on your WooCommerce checkout page in customers phone field by default based on your selected country drop-down. We have used <a href="https://restcountries.eu/" target="_blank">REST COUNTRIES API</a> to fetch country ISD code on checkout page.

This plugin is completely free with all features included so, there is no Pro version of this plugin. You have to create your account on Divine Info SMS service using <a href="http://sms.divineinfo.net/" target="_blank">this</a> link.

Credits : <a href="https://restcountries.eu/" target="_blank">REST COUNTRIES API</a>

<a href="https://www.paypal.me/anilmeena" target="_blank">Donate Link</a>

### Features

* Sends SMS notification everytime while update of order status.
* Fully Customizable SMS text for each status message
* Sending Order Details ( order no, order status, order shop name and order amount and many more shortcodes supported) in SMS text.
* Facility to view complete SMS history
* Different SMS send corresponding to different WooCommerce Order Status.
* Works with WooCommerce 2.0+.

== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==
= Where I can find plugin settings page? =
After the plugin has been activated you can find "SMS Notification" tab in WooCommerce settings link.

= Can I change SMS gateway from plugin setting ? =
No. You can't change SMS gateway since this plugin only support to Divine Info SMS service.
